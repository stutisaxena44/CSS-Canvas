# Happy 2026 🎆

This project is an **animated New Year celebration webpage** built using **HTML, CSS, and JavaScript Canvas**. It features a fullscreen fireworks animation with a glowing “2026” greeting and a festive night-sky background.

## 🎉 Project Overview

The **Happy 2026** page uses the HTML `<canvas>` element to create a dynamic fireworks system. Each firework launches from the bottom of the screen and explodes into colorful particles, creating a smooth and immersive celebration effect.

## 🧩 Key Features

- Fullscreen animated fireworks
- Smooth particle-based explosion effects
- Responsive canvas that adapts to screen size
- Modern typography using Google Fonts
- Festive glowing text overlay
- Continuous animation using `requestAnimationFrame`

## 🛠 Technologies Used

- HTML5  
- CSS3  
- JavaScript  
- Canvas API  
- Google Fonts (Montserrat)

## 🎨 Concepts Demonstrated

- Canvas rendering and animation
- Object-oriented JavaScript (classes)
- Particle systems
- Physics concepts (gravity, friction, velocity)
- Alpha blending for fade-out effects
- Responsive resizing with window events

## 🚀 How to Run

1. Save the code as `index.html`
2. Open the file in any modern web browser
3. Enjoy the animated fireworks and New Year greeting 🎇

## 🎯 Purpose of the Project

This project was created to:
- Practice Canvas-based animations
- Understand particle systems
- Explore creative coding with JavaScript
- Build visually engaging web experiences
